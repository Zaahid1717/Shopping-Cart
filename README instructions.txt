# Shopping Cart Project - Setup Instructions

## 1. Clone the Repository
```bash
git clone https://github.com/Zaahid1717/Shopping-Cart.git
cd Shopping-Cart
```

## 2. Install Dependencies
Navigate to the project folder and run the following command to install all necessary packages:
```bash
npm install
```

## 3. Required Libraries
Make sure the following key libraries are installed (these will be installed via `npm install`):
- React
- Axios
- Nodemon (for development)
- React-Scripts

## 4. Run the Project
Start the development server:
```bash
npm start
```

## 5. API Setup
- If you are fetching data from a backend, ensure your backend server is running. Modify the API endpoint in your frontend code to point to the correct server.

## 6. Development Server
To auto-restart your server while coding, you can use:
```bash
npx nodemon index.js
```

## 7. Backend Setup (if applicable)
- Ensure that your backend server (e.g., Express) is running.
- Check API URLs in your code (e.g., `http://localhost:5000/api/`).

Now your project should be running!
